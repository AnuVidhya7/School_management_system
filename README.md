School Management System
Project Overview
The School Management System is a web-based application developed using Java, JSP, Servlets, JDBC, and Oracle Database. This system allows administrators to add student records and view existing student records based on student name and admission date.
The application follows the MVC (Model–View–Controller) architecture and uses DAO and Service layers for clean separation of business logic and database operations.
Project Objective
The objective of this project is to create an online system to:
Add new student records
Store student information in a database
View existing student records
Validate user input
Maintain unique record IDs
Display success or error messages based on operations
